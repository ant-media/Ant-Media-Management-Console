Configure Auto Start on Viewer Detected and Auto Stop on No Viewer Detected

Stream Id: {{stream.streamId}}
Stream Name: {{stream.name}}
Stream Type: {{stream.type}}

If this feature is enabled, the stream will stop automatically after a specified amount of time has elapsed without any viewer being detected. Additionally, if this feature is enabled and the stream type is an IP Camera, Stream Source, VOD or playlist, the stream will automatically start when a viewer begins to play it.

This setting can be particularly useful for IP cameras, as it can help save bandwidth by automatically stopping the stream when there are no viewers, and starting it again when someone begins to watch. Please remember that recording will also stop when stream is stopped.
